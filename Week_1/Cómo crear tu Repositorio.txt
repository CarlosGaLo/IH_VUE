					COMANDOS CMD
cd.. -> Change directory, cambia de carpeta
git pull -> Hace el pull (baja los cambios del repositorio)
git add . -> add de todo, si quieres de fichero concreto entonces: git add rutaYNombreDelFichero) 
git commit -m"mensaje del commit" -> Pone un mensaje al commit
git push -> Sube los cambios al repositorio
