- Comandos.
	ls -> pinta en pantalla la ubicación (list director)
	cd -> cambiar directorio (change directory)
	cd.. -> Regresa un paso hacia atrás 
	mkdir -> Make-Directory, crea una carpeta nueva
	touch (apple) / echo (windows) -> Crea distintos tipos de archivo
	Flecha arriba / abajo -> Revisas lo escrito
	cls ó clear -> limpia la pantalla
	open -> abre un archivo determinado
